# overbuffed-gulp
<small>v. 2.1.0</small>

## Install all packages
```sh
$ npm i
```

or via yarn
```sh
$ yarn
```

## Launch compiling and live-server/live-reload
```sh
$ npm start
```

or via yarn

```sh
$ yarn start
```

## Build for production
```sh
$ npm run build
```

or via yarn

```sh
$ yarn build
```

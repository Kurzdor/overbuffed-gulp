# overbuffed-gulp
<small>v. 3.0.0</small>

## Install all packages
```sh
$ npm install
```

or via yarn
```sh
$ yarn
```

## Launch compiling and live-server/live-reload
```sh
$ npm start / npm run dev
```

or via yarn

```sh
$ yarn start / yarn dev
```

## Build for production (minify styles, scripts, images, etc.)
```sh
$ npm run build / npm run prod
```

or via yarn

```sh
$ yarn build / yarn prod
```
